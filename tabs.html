Options

Options can be set as object atributes trough javascript and/or trough markup, as attributes, or both.

In case when we use both, markup options precedes javascript.

tabs / data-tabs
Tab selector, it must be anchor tag ('a'), since href attribute content is used for destination element, which will be shown.
Default value (string): 'a'
selectedClass / data-selectedclass
CSS class, assigned to selected tab.
Default value (string): 'tab_selected'
onChange / data-onchange
Callback function called before tab show. Tab content id is passed as parameter. In javascript it can be passed anonymous function, in case when we passing function name trough atribute, we can use only function name.
Default (function or string): function(id){}
zSwitch / data-zswitch
Used when we want to change tab content visibilty by changeing zIndex, this way we never hide some element, just push it to the back. Useful for content with images, since every browser optimizes image loading, and skips loading of hidden images.
Default (bool): false
tabContent / data-tabcontent
Used with previous param, to define selector for tab content.
Default (string): '.tab_content'

$('.tabs').tabs();
